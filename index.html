Haunter Syntax

A syntax theme for Github's Atom Editor.

Installation

You can find and install it from within Atom, under Settings > Themes

A UI theme for Github's Atom Editor.

Haunter Iterm

Color theme for iterm2.

Installation

You can import haunter.itermcolors from within iterm2, under Preferences > Profiles > Colors
